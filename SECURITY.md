# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in WisprFlex, please report it privately.

**Do NOT open a public issue.**

### How to Report

Email: security@wisprflex.dev or shashankatthaluri@gmail.com (or use GitHub's private vulnerability reporting) 

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Timeline

We aim to acknowledge reports within 48 hours and provide an initial assessment within 7 days.
